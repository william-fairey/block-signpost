== Description ==

This block creates a hyperlink to insert blocks into.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/signpost` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= How to use? =

Place signpost into content use the link button in the toolbar to set a url, you can edit the url to open in a new window.
